# Nhóm phím tắt trong menu File
- Ctrl + N: Tạo file mới.
- Ctrl + O: Mở file.
- Ctrl + Alt + O: Mở file từ Adobe Bridge.
- CtrlL + W: Đóng file.
- Ctrl + S: Lưu.
- Ctrl + Shift + S: Lựa ra các dạng khác.
- Ctrl + alt + S: Lưu thêm 1 bản copy.
- Ctrl + P: In ảnh.
